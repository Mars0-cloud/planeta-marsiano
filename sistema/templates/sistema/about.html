El Proyecto

Planeta Marsiano es mi biblioteca cinematográfica personal desarrollada como proyecto web, donde documento y catalogo todas las películas que he visto con sus respectivos ratings y análisis.

Sistema de Rating Personal

Cada película tiene mi valoración personal del 1 al 5, documentando mi experiencia cinematográfica.

Catalogación por Géneros

Organización automática por géneros obtenidos de TMDb para facilitar la navegación.

Filtros Avanzados

Búsqueda por título, año, década, género y rating para encontrar exactamente lo que busco.

Estadísticas Detalladas

Análisis completo de mis hábitos cinematográficos y distribución de géneros favoritos.

Desarrollo Técnico

El proyecto está desarrollado usando tecnologías web modernas, con integración a la API de TMDb para obtener información detallada de las películas.

Tecnologías utilizadas:
Django/Python HTML5 CSS3 JavaScript Bootstrap 5 SQLite
Integraciones:
  • API de TMDb para información de películas
  • Importación automática desde Letterboxd CSV
  • Chart.js para visualización de estadísticas

Funcionalidades

Una aplicación web completa para gestionar y analizar mi experiencia cinematográfica personal.

Importación automática

Comando Django que importa mi historial desde Letterboxd y enriquece con datos de TMDb

Búsqueda y filtrado

Filtros por género, año, década, rating mínimo con actualización dinámica

Vista detallada

Página individual para cada película con sinopsis, géneros y datos completos

Dashboard estadístico

Análisis detallado con gráficos interactivos de mis patrones cinematográficos

Diseño responsivo

Interfaz adaptable para cualquier dispositivo
